border

Establece el estilo de línea para los cuatro lados del borde de un elemento.

border-left, border-right, border-top, border-bottom

Establece todas las propiedades del borde por separado de un elemento.

border-radius

Redondea las esquinas del borde exterior de un elemento.
Puede establecer un solo radio para hacer esquinas circulares o dos radios para hacer esquinas elípticas.

background-image

Establece una o más imágenes de fondo en un elemento.

color

Agrega color.
Puede ser en RGB o en hexadecimal.

font

Establece el tamaño y un tipo de fuente a los elementos.
EJEMPLO:
bold 13pt sans-serif

font-family

Especifica una lista de uno o más nombres de familia de fuentes para el elemento seleccionado.
EJEMPLO:
Papyrus, Herculanum, Party LET, Curlz MT, Harrington, fantasy.

font-style, font-weight

Establece la fuente como italicas, normales o obli
